# 🤖 Automated EDA Agent  
### GenZ for GenAI – Challenge 2 Submission

An intelligent data science agent that automates Exploratory Data Analysis (EDA), anomaly detection, feature engineering suggestions, visualization, and model recommendation using a scalable, agent-based architecture.

---

## 🚀 Project Overview

This project builds an **end-to-end automated EDA platform** that:
- Accepts CSV / SQL / Parquet datasets
- Profiles data automatically
- Generates insights and anomalies
- Recommends features and ML models
- Produces a clean HTML report for easy interpretation

The goal is to **reduce data analysis time from hours to minutes** while maintaining high insight quality.

---

## ✨ Key Features

- 📊 Automated Exploratory Data Analysis (EDA)
- 🧠 Agent-based architecture (modular & scalable)
- 🚨 Anomaly detection
- 🛠️ Feature engineering suggestions
- 🤖 AutoML model recommendation
- 📈 Automated visualizations
- 🧬 ScaleDown metadata compression (~75%)
- 📄 Auto-generated HTML & JSON reports
